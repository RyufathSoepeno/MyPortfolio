Mentcare System User Guide

Patient Management

To manage patients, navigate to the Patient Management section.

  • Add Patient: Add a new patient by filling in their full name, date of birth, birth place, gender, telephone number, email, and photo. The system will automatically generate a unique ID for the patient.
  • Patients List: View the full list of patients in the system. Each row includes a "LINK" that provides detailed patient information and the number of medications they have been prescribed.

Appointments

Manage and view all appointments in the Appointments section.

  • Add Appointment: Add a new appointment by selecting the patient's name (already stored in the database), specifying the date and time, providing a description, and selecting the type of appointment.
  • Calendar: View a calendar displaying all appointments in chronological order.

Medication Management

Manage medications through the Medication Management section.

  • Add Medication: Add a new medication by entering the medication name, formulation type, class of medication, manufacturer, and description (including details like expiry dates, prescriptions, etc.). The system will automatically generate a unique ID for the medication.
  • Medications List: View the full list of medications in the system. Each row includes a "LINK" that provides detailed medication information and the number of times it has been prescribed.
  • Assign Medication: Assign an existing medication to a patient. This will update the information found in the "LINK" sections of both the patient and medication lists.

Reports

View a comprehensive report of all activities in the system. Note: This feature is currently under development and cannot be used at this time.
